# Dizzy's Disease - Complete Game Application

## Quick Start

### Game (Godot)
1. Install Godot 4.x
2. Open `project.godot`
3. Run project

### API Backend
1. `cd api`
2. `pip install -r requirements.txt`
3. `python main.py`

### Local Testing
Run `./start_test_environment.sh`
